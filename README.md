# Michigan State Roleplay Website

Static landing page for Michigan State Roleplay.

## Files

- `index.html` - main website page
- `styles.css` - site styling
- `script.js` - mobile menu interaction
- `logo.svg` - custom logo

## Release on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Make sure the default branch is named `main`.
4. In GitHub, open `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `GitHub Actions`.
6. Push to `main`.
7. Wait for the `Deploy static site to GitHub Pages` action to finish.
8. Your site will be published at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Quick Git Commands

If Git is installed on your PC, you can run these in this folder:

```powershell
git init
git branch -M main
git add .
git commit -m "Initial website release"
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPOSITORY-NAME.git
git push -u origin main
```
